Техническое задание веб-проекта «Cibo».

1. Общие сведения.
Проект «Cibo» — это веб-платформа для обмена кулинарными рецептами и формирования сообщества любителей кулинарии. Сайт предназначен для пользователей с разным уровнем опыта: от новичков до профессиональных поваров. Основная задача — создать удобное, безопасное и функциональное пространство для публикации, поиска и обсуждения рецептов.

2. Цели и задачи проекта.
Цель: создание современного веб-приложения для обмена кулинарными знаниями и рецептами.

Задачи:
- Реализовать структурированный каталог рецептов с возможностью поиска и фильтрации.
- Обеспечить функционал публикации пользовательских рецептов с поддержкой изображений и подробных описаний.
- Внедрить систему управления контентом: редактирование и удаление собственных рецептов.
- Организовать персонализированные профили пользователей с возможностью редактирования данных.
- Реализовать надёжную систему регистрации и авторизации для защиты персональных данных.

3. Применяемые технологии.
Проект разрабатывается на языке Python с использованием фреймворка Flask. Для управления зависимостями используется файл requirements.txt.

В проекте реализованы следующие технологии:
- Веб-интерфейс: используется Bootstrap для адаптивного и современного дизайна.
- ORM-модели: применяются для работы с базой данных (SQLAlchemy).
- Регистрация и авторизация: реализована система безопасного входа и управления аккаунтами пользователей.
- Загрузка файлов: поддерживается загрузка аватара профиля и изображений к рецептам.
- API: реализован REST API для взаимодействия клиентской и серверной частей.
- Хранение данных: используется реляционная база данных (SQLite).
- Хостинг: проект будет размещён на облачной платформе или веб-хостинге с поддержкой Python/Flask.

4. Функциональные требования.

4.1. Пользовательский путь.
1.Регистрация/авторизация:
   - Ввод логина, пароля, e-mail.

2.Каталог рецептов:
   - Поиск по названию.
   - Просмотр подробной информации о рецепте (ингредиенты, шаги, фото).

3.Публикация рецептов:
   - Добавление нового рецепта с фото, описанием, ингредиентами и инструкциями.
   - Автоматическая привязка рецепта к автору.

4.Управление контентом:
   - Редактирование и удаление собственных рецептов.

5.Редактирование профиля:
   - Изменение личных данных, пароля, аватара.

4.2. Администрирование.
- Управление пользователями (при необходимости).
- Модерация контента.

5.Безопасность.
- Все данные пользователей хранятся в зашифрованном виде.
- Пароли хэшируются перед сохранением в базе данных.

6. Хостинг.
Проект будет развёрнут на хостинге с поддержкой Python и Flask (Heroku). Необходимо обеспечить работоспособность сайта 24/7, а также возможность масштабирования при увеличении числа пользователей.

7. Заключение.
Проект «Cibo» — это современная социальная платформа для любителей кулинарии, реализованная на Python/Flask с использованием всех необходимых технологий: ORM, REST API, Bootstrap, система авторизации, загрузка файлов и база данных. Сайт обеспечивает удобство, безопасность и масштабируемость, а также способствует развитию гастрономической культуры и формированию сообщества единомышленников.
